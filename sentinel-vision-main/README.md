# Sentinel Vision

React dashboard for the Suspicious Activity Detection system — live CCTV, alerts, analytics, and security notifications.

## Run

```bash
npm install
npm run dev
```

Open http://localhost:8080 and sign in with admin credentials from `backend/.env`.

## Environment

Copy `.env.example` to `.env` and set `VITE_API_BASE_URL`, Supabase keys.
